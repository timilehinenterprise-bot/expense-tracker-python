# Expense Tracker (Python)

A simple Python application to track daily expenses and monitor spending habits.

## 🚀 Features
- Add expenses with name and amount
- View all expenses
- Delete expenses
- Calculate total spending

## 🛠️ Tech Used
- Python

## ▶️ How to Run
python expert_tracker.py

## 📌 Future Improvements

- Add categories (food, transport, etc.)
